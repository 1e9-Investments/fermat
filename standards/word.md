# Investment Research Document Formatting Standard

**Why:** Institutional-grade documents require consistent, dense formatting that maximizes information density and signals professionalism.

**How to apply:** Apply these rules to ALL Word documents generated for investment research - PIBs, CIMs, memos, tearsheets.

---

## Page & Margins
- US Letter size
- All margins 0.5in
- No gutter
- Content area maximized

## Typography

### Primary Font
- **Aptos Narrow** — primary body font
- **Calibri** — fallback/theme default
- Apply consistently across all elements (body, bullets, tables, headers, footers)
- Never mix fonts

### Body Text
- 11pt
- Line spacing exactly 1.0x
- Paragraph spacing 6pt before / 6pt after
- Left-aligned
- Never use blank paragraphs or blank lines as spacers — all spacing through style-defined before/after values only

## Heading Hierarchy (Greenish Palette - Wall St Standard)

| Level | Style | Color | Property |
|-------|-------|-------|----------|
| H1 | Bold + Underline | #1A3C2B deep forest green | keepNext |
| H2 | Bold only | #2D6A4F mid forest green | keepNext |
| H3 | Bold Italic | #40916C medium green | keepNext |
| Sub-labels | Italic only, no bold | #52796F muted sage | keepNext |

Visual logic: each level lightens by one step; underline reserved for H1 only as primary anchor

## Bullets
- Two levels maximum
- Level 0: • at 0.5in left, 0.25in hanging
- Level 1: ○ at 1.0in left, 0.25in hanging
- All bullets use 6pt before / 6pt after (including executive summary)
- Never a third level
- Never blank lines between bullets

## Tables

### Font & Sizing
- 11pt theme default (Aptos Narrow / Calibri)
- Reduce to 10pt only when required for page fit
- Apply size consistently across entire table — never mixed sizes within one table

### Cell Formatting
- Line spacing inside cells: 0pt (exact) — no leading, maximum density
- Cell padding: tight (default Word minimum — no extra top/bottom)
- Figures: right-aligned
- Labels: left-aligned

### Column Widths
- Manually set, never autofit
- Distribute proportionally among similar data types (e.g., all currency columns equal width)
- Width precision critical — columns must sum exactly to content area

### Borders & Shading
- Borders: 0.5pt single, light gray #F2F2F2 — near-invisible, clean
- Row shading: alternating white / light gray #F5F5F5
- Key metric rows (Revenue, PAT, Total Debt, Total Assets): highlighted sage #C2D69B
- Single most critical row: bolded

### Header Row
- Fill: dark forest green #1F4E36
- Text: white, bold, 11pt (or 10pt if table reduced)

### Table Labels & Bullets
- Table label (italic sub-label style) glued to table above via keepNext — zero visual gap
- Every data table followed by 2-4 analytical bullets
- 6pt spacing between table bottom and first bullet

## Spacing Discipline
- Body → body: 6pt
- Body → table: 6pt
- Table → post-table bullets: 6pt
- Bullet → bullet: 6pt
- Heading keepNext prevents orphaned headers and labels
- No decorative whitespace anywhere

## Sources Line
Format: `Sources: [Primary]; [Source 1]; [Source 2].`
- Primary source: "20in20 Research" (for 20in20 Partners) or "1e9 Investments" (for 1e9) — always first
- Placement: immediately below last post-table bullet or section close
- Style: italic, 9pt, color #595959

## Writing Style
- Dense, telegraphic, institutional analyst voice
- No padding or transitions
- Every sentence carries a data point or analytical judgment
- Conclusions stated firm
- Problems named plainly ("technically insolvent," "permanently rolling")
- Parenthetical elaboration packs context inline
- **Bold the single most important number or phrase per bullet**

## Punctuation
- Never use long dashes (em dashes) — use short dashes (hyphens) instead

## Financial Figures
- USD pair with VND: US$Xmm (c.VNDXbn)
- Billions: bn | Trillions: tn
- Ratios: Xx
- Negatives in tables: (VNDXbn) | Negatives in prose: -VNDXbn
- Growth: +X% YoY
- Estimates: prefix ~ or c.
- Year labels: 2025A / 2026F

## Abbreviations & Names
- Define on first use in parentheses, then use abbreviated form consistently
- Preserve Vietnamese diacritics in all proper nouns (Nguyễn, Thủ Thiêm, etc.)

## General Principles
- No filler
- No hedging unless analytically justified
- Write for sophisticated institutional audience that penalizes vagueness
